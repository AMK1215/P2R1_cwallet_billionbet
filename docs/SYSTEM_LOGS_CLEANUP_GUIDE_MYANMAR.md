# рђЁрђћрђЁрђ║рђюрђ▒рђгрђирђЎрђ╗рђгрђИрђћрђЙрђёрђирђ║ рђЏрђЙрђёрђ║рђИрђюрђёрђ║рђИрђЎрђЙрђ»рђЁрђћрђЁрђ║ рђАрђърђ»рђХрђИрђЋрђ╝рђ»рђърђ░рђюрђЎрђ║рђИрђірђйрђЙрђћрђ║
# (System Logs and Cleanup System User Guide)

## ­ЪЊІ рђЁрђгрђЏрђёрђ║рђИрђЋрђФрђЮрђёрђ║рђЎрђЙрђ»рђЎрђ╗рђгрђИ (Table of Contents)

1. [рђЁрђћрђЁрђ║рђюрђ▒рђгрђирђЎрђ╗рђгрђИрђАрђђрђ╝рђ▒рђгрђёрђ║рђИ (About System Logs)](#рђЁрђћрђЁрђ║рђюрђ▒рђгрђирђЎрђ╗рђгрђИрђАрђђрђ╝рђ▒рђгрђёрђ║рђИ)
2. [рђёрђйрђ▒рђђрђ╝рђ▒рђИрђюрђ»рђЋрђ║рђєрђ▒рђгрђёрђ║рђЂрђ╗рђђрђ║рђюрђ▒рђгрђирђЎрђ╗рђгрђИ (Financial Transaction Logs)](#рђёрђйрђ▒рђђрђ╝рђ▒рђИрђюрђ»рђЋрђ║рђєрђ▒рђгрђёрђ║рђЂрђ╗рђђрђ║рђюрђ▒рђгрђирђЎрђ╗рђгрђИ)
3. [рђЮрђўрђ║рђЪрђйрђљрђ║рђюрђ▒рђгрђирђЎрђ╗рђгрђИ (Webhook Logs)](#рђЮрђўрђ║рђЪрђйрђљрђ║рђюрђ▒рђгрђирђЎрђ╗рђгрђИ)
4. [рђњрђђрђ║рђњрђюрђ▒рђгрђирђЎрђ╗рђгрђИ (Deadlock Logs)](#рђњрђђрђ║рђњрђюрђ▒рђгрђирђЎрђ╗рђгрђИ)
5. [рђЁрђћрђЁрђ║рђюрђ▒рђгрђирђЎрђ╗рђгрђИрђђрђГрђ» рђЏрђЙрђёрђ║рђИрђюрђёрђ║рђИрђЂрђ╝рђёрђ║рђИ (System Logs Cleanup)](#рђЁрђћрђЁрђ║рђюрђ▒рђгрђирђЎрђ╗рђгрђИрђђрђГрђ»-рђЏрђЙрђёрђ║рђИрђюрђёрђ║рђИрђЂрђ╝рђёрђ║рђИ)
6. [рђАрђЎрђ╝рђћрђ║рђюрђ»рђЋрђ║рђєрђ▒рђгрђёрђ║рђЂрђ╗рђђрђ║рђЎрђ╗рђгрђИ (Quick Actions)](#рђАрђЎрђ╝рђћрђ║рђюрђ»рђЋрђ║рђєрђ▒рђгрђёрђ║рђЂрђ╗рђђрђ║рђЎрђ╗рђгрђИ)

## рђЁрђћрђЁрђ║рђюрђ▒рђгрђирђЎрђ╗рђгрђИрђАрђђрђ╝рђ▒рђгрђёрђ║рђИ (About System Logs)

### ­Ъј» рђЁрђћрђЁрђ║рђюрђ▒рђгрђирђЎрђ╗рђгрђИрЂЈрђЏрђірђ║рђЏрђйрђџрђ║рђЂрђ╗рђђрђ║ (Purpose of System Logs)
рђЁрђћрђЁрђ║рђюрђ▒рђгрђирђЎрђ╗рђгрђИрђърђірђ║ рђѓрђГрђЎрђ║рђИрђЋрђюрђђрђ║рђќрђ▒рђгрђёрђ║рђИрЂЈ рђюрђ»рђЋрђ║рђєрђ▒рђгрђёрђ║рђЂрђ╗рђђрђ║рђЎрђ╗рђгрђИрђђрђГрђ» рђЎрђЙрђљрђ║рђљрђЎрђ║рђИрђљрђёрђ║рђЉрђгрђИрђърђ▒рђг рђњрђ▒рђљрђгрђЎрђ╗рђгрђИрђќрђ╝рђЁрђ║рђЋрђФрђърђірђ║рЂІ

### ­ЪћД рђЁрђћрђЁрђ║рђюрђ▒рђгрђирђЎрђ╗рђгрђИрЂЈрђАрђЊрђГрђђрђюрђ»рђЋрђ║рђєрђ▒рђгрђёрђ║рђЂрђ╗рђђрђ║рђЎрђ╗рђгрђИ (Main Functions)
- **рђюрђ»рђЋрђ║рђєрђ▒рђгрђёрђ║рђЂрђ╗рђђрђ║рђЎрђ╗рђгрђИрђђрђГрђ» рђЎрђЙрђљрђ║рђљрђЎрђ║рђИрђљрђёрђ║рђЂрђ╝рђёрђ║рђИ** - рђЁрђћрђЁрђ║рЂЈ рђюрђ»рђЋрђ║рђєрђ▒рђгрђёрђ║рђЂрђ╗рђђрђ║рђЎрђ╗рђгрђИрђђрђГрђ» рђЎрђЙрђљрђ║рђљрђЎрђ║рђИрђљрђёрђ║рђЂрђ╝рђёрђ║рђИ
- **рђЋрђ╝рђ┐рђћрђгрђЎрђ╗рђгрђИрђђрђГрђ» рђЏрђЙрђгрђќрђйрђ▒рђЂрђ╝рђёрђ║рђИ** - рђЁрђћрђЁрђ║рђЋрђ╝рђ┐рђћрђгрђЎрђ╗рђгрђИрђђрђГрђ» рђЏрђЙрђгрђќрђйрђ▒рђЂрђ╝рђёрђ║рђИ
- **рђЁрђйрђЎрђ║рђИрђєрђ▒рђгрђёрђ║рђЏрђірђ║рђђрђГрђ» рђЁрђ▒рђгрђёрђирђ║рђђрђ╝рђірђирђ║рђЂрђ╝рђёрђ║рђИ** - рђЁрђћрђЁрђ║рЂЈ рђЁрђйрђЎрђ║рђИрђєрђ▒рђгрђёрђ║рђЏрђірђ║рђђрђГрђ» рђЁрђ▒рђгрђёрђирђ║рђђрђ╝рђірђирђ║рђЂрђ╝рђёрђ║рђИ
- **рђюрђ»рђХрђЂрђ╝рђ»рђХрђЏрђ▒рђИрђђрђГрђ» рђЁрђ▒рђгрђёрђирђ║рђђрђ╝рђірђирђ║рђЂрђ╝рђёрђ║рђИ** - рђЁрђћрђЁрђ║рЂЈ рђюрђ»рђХрђЂрђ╝рђ»рђХрђЏрђ▒рђИрђђрђГрђ» рђЁрђ▒рђгрђёрђирђ║рђђрђ╝рђірђирђ║рђЂрђ╝рђёрђ║рђИ

## рђёрђйрђ▒рђђрђ╝рђ▒рђИрђюрђ»рђЋрђ║рђєрђ▒рђгрђёрђ║рђЂрђ╗рђђрђ║рђюрђ▒рђгрђирђЎрђ╗рђгрђИ (Financial Transaction Logs)

### ­Ъњ░ рђёрђйрђ▒рђђрђ╝рђ▒рђИрђюрђ»рђЋрђ║рђєрђ▒рђгрђёрђ║рђЂрђ╗рђђрђ║рђюрђ▒рђгрђирђЎрђ╗рђгрђИрЂЈрђАрђЎрђ╗рђГрђ»рђИрђАрђЁрђгрђИрђЎрђ╗рђгрђИ (Types of Financial Transaction Logs)

#### 1. **рђёрђйрђ▒рђърђйрђёрђ║рђИрђЎрђЙрђ»рђЎрђ╗рђгрђИ (Deposits)**
- **рђёрђйрђ▒рђърђйрђёрђ║рђИрђюрђ»рђЋрђ║рђєрђ▒рђгрђёрђ║рђЂрђ╗рђђрђ║рђЎрђ╗рђгрђИ** - рђђрђЁрђгрђИрђърђЎрђгрђИрђЎрђ╗рђгрђИрЂЈ рђёрђйрђ▒рђърђйрђёрђ║рђИрђюрђ»рђЋрђ║рђєрђ▒рђгрђёрђ║рђЂрђ╗рђђрђ║рђЎрђ╗рђгрђИ
- **рђёрђйрђ▒рђърђйрђёрђ║рђИрђАрђљрђірђ║рђЋрђ╝рђ»рђЂрђ╝рђёрђ║рђИ** - рђёрђйрђ▒рђърђйрђёрђ║рђИрђЎрђЙрђ»рђЎрђ╗рђгрђИрђђрђГрђ» рђАрђљрђірђ║рђЋрђ╝рђ»рђЂрђ╝рђёрђ║рђИ
- **рђёрђйрђ▒рђърђйрђёрђ║рђИрђёрђ╝рђёрђ║рђИрђЋрђџрђ║рђЂрђ╝рђёрђ║рђИ** - рђёрђйрђ▒рђърђйрђёрђ║рђИрђЎрђЙрђ»рђЎрђ╗рђгрђИрђђрђГрђ» рђёрђ╝рђёрђ║рђИрђЋрђџрђ║рђЂрђ╝рђёрђ║рђИ

#### 2. **рђёрђйрђ▒рђЉрђ»рђљрђ║рђЎрђЙрђ»рђЎрђ╗рђгрђИ (Withdrawals)**
- **рђёрђйрђ▒рђЉрђ»рђљрђ║рђюрђ»рђЋрђ║рђєрђ▒рђгрђёрђ║рђЂрђ╗рђђрђ║рђЎрђ╗рђгрђИ** - рђђрђЁрђгрђИрђърђЎрђгрђИрђЎрђ╗рђгрђИрЂЈ рђёрђйрђ▒рђЉрђ»рђљрђ║рђюрђ»рђЋрђ║рђєрђ▒рђгрђёрђ║рђЂрђ╗рђђрђ║рђЎрђ╗рђгрђИ
- **рђёрђйрђ▒рђЉрђ»рђљрђ║рђАрђљрђірђ║рђЋрђ╝рђ»рђЂрђ╝рђёрђ║рђИ** - рђёрђйрђ▒рђЉрђ»рђљрђ║рђЎрђЙрђ»рђЎрђ╗рђгрђИрђђрђГрђ» рђАрђљрђірђ║рђЋрђ╝рђ»рђЂрђ╝рђёрђ║рђИ
- **рђёрђйрђ▒рђЉрђ»рђљрђ║рђёрђ╝рђёрђ║рђИрђЋрђџрђ║рђЂрђ╝рђёрђ║рђИ** - рђёрђйрђ▒рђЉрђ»рђљрђ║рђЎрђЙрђ»рђЎрђ╗рђгрђИрђђрђГрђ» рђёрђ╝рђёрђ║рђИрђЋрђџрђ║рђЂрђ╝рђёрђ║рђИ

#### 3. **рђюрђйрђЙрђ▓рђЋрђ╝рђ▒рђгрђёрђ║рђИрђЎрђЙрђ»рђЎрђ╗рђгрђИ (Transfers)**
- **рђАрђ▒рђѓрђ╗рђёрђирђ║рђЎрђ╗рђгрђИрђАрђђрђ╝рђгрђИ рђюрђйрђЙрђ▓рђЋрђ╝рђ▒рђгрђёрђ║рђИрђЎрђЙрђ»рђЎрђ╗рђгрђИ** - рђАрђ▒рђѓрђ╗рђёрђирђ║рђЎрђ╗рђгрђИрђАрђђрђ╝рђгрђИ рђёрђйрђ▒рђюрђйрђЙрђ▓рђЋрђ╝рђ▒рђгрђёрђ║рђИрђЎрђЙрђ»рђЎрђ╗рђгрђИ
- **рђђрђЁрђгрђИрђърђЎрђгрђИрђЎрђ╗рђгрђИрђАрђђрђ╝рђгрђИ рђюрђйрђЙрђ▓рђЋрђ╝рђ▒рђгрђёрђ║рђИрђЎрђЙрђ»рђЎрђ╗рђгрђИ** - рђђрђЁрђгрђИрђърђЎрђгрђИрђЎрђ╗рђгрђИрђАрђђрђ╝рђгрђИ рђёрђйрђ▒рђюрђйрђЙрђ▓рђЋрђ╝рђ▒рђгрђёрђ║рђИрђЎрђЙрђ»рђЎрђ╗рђгрђИ

### ­ЪћД рђёрђйрђ▒рђђрђ╝рђ▒рђИрђюрђ»рђЋрђ║рђєрђ▒рђгрђёрђ║рђЂрђ╗рђђрђ║рђюрђ▒рђгрђирђЎрђ╗рђгрђИрђђрђГрђ» рђђрђ╝рђірђирђ║рђЏрђЙрђ»рђЂрђ╝рђёрђ║рђИ (Viewing Financial Transaction Logs)

#### **рђюрђ▒рђгрђирђЎрђ╗рђгрђИрђђрђГрђ» рђђрђ╝рђірђирђ║рђЏрђЙрђ»рђЂрђ╝рђёрђ║рђИ (Viewing Logs)**
1. **"рђЁрђћрђЁрђ║рђюрђ▒рђгрђирђЎрђ╗рђгрђИ" рђЎрђ«рђћрђ░рђИрђђрђГрђ» рђћрђЙрђГрђЋрђ║рђЋрђФ**
2. **"рђёрђйрђ▒рђђрђ╝рђ▒рђИрђюрђ»рђЋрђ║рђєрђ▒рђгрђёрђ║рђЂрђ╗рђђрђ║рђюрђ▒рђгрђирђЎрђ╗рђгрђИ" рђђрђГрђ» рђћрђЙрђГрђЋрђ║рђЋрђФ**
3. **рђюрђГрђ»рђАрђЋрђ║рђърђ▒рђг рђюрђ▒рђгрђирђЎрђ╗рђгрђИрђђрђГрђ» рђЏрђЙрђгрђќрђйрђ▒рђЋрђФ**
4. **рђюрђ▒рђгрђирђАрђърђ▒рђИрђЁрђГрђљрђ║рђђрђГрђ» рђђрђ╝рђірђирђ║рђЏрђЙрђ»рђЋрђФ**

#### **рђюрђ▒рђгрђирђЎрђ╗рђгрђИрђђрђГрђ» рђЁрђ»рђљрђ║рђЉрђ»рђљрђ║рђЂрђ╝рђёрђ║рђИ (Exporting Logs)**
1. **рђЁрђ»рђљрђ║рђЉрђ»рђљрђ║рђюрђГрђ»рђърђ▒рђг рђюрђ▒рђгрђирђЎрђ╗рђгрђИрђђрђГрђ» рђЏрђйрђ▒рђИрђЂрђ╗рђџрђ║рђЋрђФ**
2. **"CSV рђЁрђ»рђљрђ║рђЉрђ»рђљрђ║рђЏрђћрђ║" рђЂрђюрђ»рђљрђ║рђђрђГрђ» рђћрђЙрђГрђЋрђ║рђЋрђФ**
3. **рђќрђГрђ»рђёрђ║рђђрђГрђ» рђњрђ▒рђФрђёрђ║рђИрђюрђ»рђњрђ║рђюрђ»рђЋрђ║рђЋрђФ**

## рђЮрђўрђ║рђЪрђйрђљрђ║рђюрђ▒рђгрђирђЎрђ╗рђгрђИ (Webhook Logs)

### ­ЪћЌ рђЮрђўрђ║рђЪрђйрђљрђ║рђюрђ▒рђгрђирђЎрђ╗рђгрђИрЂЈрђАрђЎрђ╗рђГрђ»рђИрђАрђЁрђгрђИрђЎрђ╗рђгрђИ (Types of Webhook Logs)

#### 1. **рђѓрђГрђЎрђ║рђИрђЋрђЏрђГрђ»рђЌрђГрђ»рђёрђ║рђњрђФрђЎрђ╗рђгрђИрђћрђЙрђёрђирђ║ рђєрђђрђ║рђърђйрђџрђ║рђЎрђЙрђ»рђЎрђ╗рђгрђИ (Game Provider Communications)**
- **GSC Plus** - GSC Plus рђѓрђГрђЎрђ║рђИрђЋрђЏрђГрђ»рђЌрђГрђ»рђёрђ║рђњрђФрђћрђЙрђёрђирђ║ рђєрђђрђ║рђърђйрђџрђ║рђЎрђЙрђ»рђЎрђ╗рђгрђИ
- **Shan Games** - Shan Games рђѓрђГрђЎрђ║рђИрђЋрђЏрђГрђ»рђЌрђГрђ»рђёрђ║рђњрђФрђћрђЙрђёрђирђ║ рђєрђђрђ║рђърђйрђџрђ║рђЎрђЙрђ»рђЎрђ╗рђгрђИ
- **Seamless** - Seamless рђѓрђГрђЎрђ║рђИрђЋрђЏрђГрђ»рђЌрђГрђ»рђёрђ║рђњрђФрђћрђЙрђёрђирђ║ рђєрђђрђ║рђърђйрђџрђ║рђЎрђЙрђ»рђЎрђ╗рђгрђИ

#### 2. **рђАрђЎрђЙрђгрђИрђЎрђ╗рђгрђИ (Errors)**
- **рђЮрђўрђ║рђЪрђйрђљрђ║рђАрђЎрђЙрђгрђИрђЎрђ╗рђгрђИ** - рђЮрђўрђ║рђЪрђйрђљрђ║рђАрђЎрђЙрђгрђИрђЎрђ╗рђгрђИрЂЈ рђЎрђЙрђљрђ║рђљрђЎрђ║рђИрђЎрђ╗рђгрђИ
- **рђЋрђ╝рђћрђ║рђюрђірђ║рђђрђ╝рђГрђ»рђИрђЁрђгрђИрђЎрђЙрђ»рђЎрђ╗рђгрђИ** - рђЋрђ╝рђћрђ║рђюрђірђ║рђђрђ╝рђГрђ»рђИрђЁрђгрђИрђЎрђЙрђ»рђЎрђ╗рђгрђИрЂЈ рђЎрђЙрђљрђ║рђљрђЎрђ║рђИрђЎрђ╗рђгрђИ

### ­ЪћД рђЮрђўрђ║рђЪрђйрђљрђ║рђюрђ▒рђгрђирђЎрђ╗рђгрђИрђђрђГрђ» рђЁрђ«рђЎрђХрђЂрђћрђирђ║рђЂрђйрђ▓рђЂрђ╝рђёрђ║рђИ (Managing Webhook Logs)

#### **рђАрђЎрђЙрђгрђИрђЎрђ╗рђгрђИрђђрђГрђ» рђЋрђ╝рђћрђ║рђюрђірђ║рђђрђ╝рђГрђ»рђИрђЁрђгрђИрђЂрђ╝рђёрђ║рђИ (Retrying Errors)**
1. **"рђЮрђўрђ║рђЪрђйрђљрђ║рђюрђ▒рђгрђирђЎрђ╗рђгрђИ" рђЎрђ«рђћрђ░рђИрђђрђГрђ» рђћрђЙрђГрђЋрђ║рђЋрђФ**
2. **рђЋрђ╝рђћрђ║рђюрђірђ║рђђрђ╝рђГрђ»рђИрђЁрђгрђИрђюрђГрђ»рђърђ▒рђг рђАрђЎрђЙрђгрђИрђђрђГрђ» рђЏрђйрђ▒рђИрђЂрђ╗рђџрђ║рђЋрђФ**
3. **"рђЋрђ╝рђћрђ║рђюрђірђ║рђђрђ╝рђГрђ»рђИрђЁрђгрђИрђЏрђћрђ║" рђЂрђюрђ»рђљрђ║рђђрђГрђ» рђћрђЙрђГрђЋрђ║рђЋрђФ**
4. **рђЋрђ╝рђћрђ║рђюрђірђ║рђђрђ╝рђГрђ»рђИрђЁрђгрђИрђЎрђЙрђ»рђђрђГрђ» рђАрђљрђірђ║рђЋрђ╝рђ»рђЋрђФ**

#### **рђЮрђўрђ║рђЪрђйрђљрђ║рђЎрђ╗рђгрђИрђђрђГрђ» рђЁрђЎрђ║рђИрђърђЋрђ║рђЂрђ╝рђёрђ║рђИ (Testing Webhooks)**
1. **рђЁрђЎрђ║рђИрђърђЋрђ║рђюрђГрђ»рђърђ▒рђг рђЮрђўрђ║рђЪрђйрђљрђ║рђђрђГрђ» рђЏрђйрђ▒рђИрђЂрђ╗рђџрђ║рђЋрђФ**
2. **"рђЁрђЎрђ║рђИрђърђЋрђ║рђЏрђћрђ║" рђЂрђюрђ»рђљрђ║рђђрђГрђ» рђћрђЙрђГрђЋрђ║рђЋрђФ**
3. **рђЁрђЎрђ║рђИрђърђЋрђ║рђЎрђЙрђ»рђЏрђюрђњрђ║рђђрђГрђ» рђђрђ╝рђірђирђ║рђЏрђЙрђ»рђЋрђФ**

## рђњрђђрђ║рђњрђюрђ▒рђгрђирђЎрђ╗рђгрђИ (Deadlock Logs)

### ­Ъћњ рђњрђђрђ║рђњрђюрђ▒рђгрђирђЎрђ╗рђгрђИрЂЈрђАрђђрђ╝рђ▒рђгрђёрђ║рђИ (About Deadlocks)

#### **рђњрђђрђ║рђњрђюрђ▒рђгрђирђєрђГрђ»рђљрђг рђўрђгрђюрђ▓? (What is a Deadlock?)**
рђњрђђрђ║рђњрђюрђ▒рђгрђирђърђірђ║ PostgreSQL рђњрђ▒рђљрђгрђўрђ▒рђирђЁрђ║рђљрђйрђёрђ║ рђќрђ╝рђЁрђ║рђЋрђйрђгрђИрђърђ▒рђг рђЋрђ╝рђ┐рђћрђгрђљрђЁрђ║рђЂрђ»рђќрђ╝рђЁрђ║рђЋрђФрђърђірђ║рЂІ рђњрђ▒рђљрђгрђЎрђ╗рђгрђИрђђрђГрђ» рђюрђ▒рђгрђирђЂрђ║рђюрђ»рђЋрђ║рђЉрђгрђИрђърђ▒рђг рђАрђЂрђ╝рђ▒рђАрђћрђ▒рђљрђйрђёрђ║ рђќрђ╝рђЁрђ║рђЋрђйрђгрђИрђЋрђФрђърђірђ║рЂІ

#### **рђњрђђрђ║рђњрђюрђ▒рђгрђирђЎрђ╗рђгрђИрЂЈрђАрђЎрђ╗рђГрђ»рђИрђАрђЁрђгрђИрђЎрђ╗рђгрђИ (Types of Deadlocks)**
- **Row Level Deadlocks** - рђАрђљрђћрђ║рђИрђАрђєрђёрђирђ║ рђњрђђрђ║рђњрђюрђ▒рђгрђирђЎрђ╗рђгрђИ
- **Table Level Deadlocks** - рђЄрђџрђгрђИрђАрђєрђёрђирђ║ рђњрђђрђ║рђњрђюрђ▒рђгрђирђЎрђ╗рђгрђИ
- **Transaction Deadlocks** - рђюрђ»рђЋрђ║рђєрђ▒рђгрђёрђ║рђЂрђ╗рђђрђ║рђАрђєрђёрђирђ║ рђњрђђрђ║рђњрђюрђ▒рђгрђирђЎрђ╗рђгрђИ

### ­ЪћД рђњрђђрђ║рђњрђюрђ▒рђгрђирђЎрђ╗рђгрђИрђђрђГрђ» рђЁрђ«рђЎрђХрђЂрђћрђирђ║рђЂрђйрђ▓рђЂрђ╝рђёрђ║рђИ (Managing Deadlocks)

#### **рђњрђђрђ║рђњрђюрђ▒рђгрђирђЎрђ╗рђгрђИрђђрђГрђ» рђђрђ╝рђірђирђ║рђЏрђЙрђ»рђЂрђ╝рђёрђ║рђИ (Viewing Deadlocks)**
1. **"рђЁрђћрђЁрђ║рђюрђ▒рђгрђирђЎрђ╗рђгрђИ" рђЎрђ«рђћрђ░рђИрђђрђГрђ» рђћрђЙрђГрђЋрђ║рђЋрђФ**
2. **"рђњрђђрђ║рђњрђюрђ▒рђгрђирђЎрђ╗рђгрђИ" рђђрђГрђ» рђћрђЙрђГрђЋрђ║рђЋрђФ**
3. **рђњрђђрђ║рђњрђюрђ▒рђгрђирђЎрђ╗рђгрђИрђђрђГрђ» рђђрђ╝рђірђирђ║рђЏрђЙрђ»рђЋрђФ**

#### **рђњрђђрђ║рђњрђюрђ▒рђгрђирђЎрђ╗рђгрђИрђђрђГрђ» рђЋрђ╝рђ▒рђюрђірђ║рђАрђ▒рђгрђёрђ║ рђюрђ»рђЋрђ║рђЂрђ╝рђёрђ║рђИ (Resolving Deadlocks)**
1. **рђЋрђ╝рђ▒рђюрђірђ║рђАрђ▒рђгрђёрђ║ рђюрђ»рђЋрђ║рђюрђГрђ»рђърђ▒рђг рђњрђђрђ║рђњрђюрђ▒рђгрђирђђрђГрђ» рђЏрђйрђ▒рђИрђЂрђ╗рђџрђ║рђЋрђФ**
2. **"рђЋрђ╝рђ▒рђюрђірђ║рђАрђ▒рђгрђёрђ║ рђюрђ»рђЋрђ║рђЏрђћрђ║" рђЂрђюрђ»рђљрђ║рђђрђГрђ» рђћрђЙрђГрђЋрђ║рђЋрђФ**
3. **рђЋрђ╝рђ▒рђюрђірђ║рђАрђ▒рђгрђёрђ║ рђюрђ»рђЋрђ║рђЎрђЙрђ»рђђрђГрђ» рђАрђљрђірђ║рђЋрђ╝рђ»рђЋрђФ**

#### **рђњрђђрђ║рђњрђюрђ▒рђгрђирђЎрђ╗рђгрђИрђђрђГрђ» рђЁрђ▒рђгрђёрђирђ║рђђрђ╝рђірђирђ║рђЂрђ╝рђёрђ║рђИ (Monitoring Deadlocks)**
1. **"рђњрђђрђ║рђњрђюрђ▒рђгрђирђЎрђ╗рђгрђИрђђрђГрђ» рђЁрђ▒рђгрђёрђирђ║рђђрђ╝рђірђирђ║рђЏрђћрђ║" рђЂрђюрђ»рђљрђ║рђђрђГрђ» рђћрђЙрђГрђЋрђ║рђЋрђФ**
2. **рђЁрђ▒рђгрђёрђирђ║рђђрђ╝рђірђирђ║рђЎрђЙрђ»рђЏрђюрђњрђ║рђђрђГрђ» рђђрђ╝рђірђирђ║рђЏрђЙрђ»рђЋрђФ**

## рђЁрђћрђЁрђ║рђюрђ▒рђгрђирђЎрђ╗рђгрђИрђђрђГрђ» рђЏрђЙрђёрђ║рђИрђюрђёрђ║рђИрђЂрђ╝рђёрђ║рђИ (System Logs Cleanup)

### ­ЪД╣ рђЏрђЙрђёрђ║рђИрђюрђёрђ║рђИрђЎрђЙрђ»рЂЈрђЏрђірђ║рђЏрђйрђџрђ║рђЂрђ╗рђђрђ║ (Purpose of Cleanup)

#### **рђўрђгрђђрђ╝рђ▒рђгрђёрђирђ║ рђЏрђЙрђёрђ║рђИрђюрђёрђ║рђИрђЏрђърђюрђ▓? (Why Cleanup?)**
- **рђњрђ▒рђљрђгрђўрђ▒рђирђЁрђ║рђЁрђйрђЎрђ║рђИрђєрђ▒рђгрђёрђ║рђЏрђірђ║** - рђњрђ▒рђљрђгрђўрђ▒рђирђЁрђ║рЂЈ рђЁрђйрђЎрђ║рђИрђєрђ▒рђгрђёрђ║рђЏрђірђ║рђђрђГрђ» рђЎрђ╝рђЙрђёрђирђ║рђљрђёрђ║рђЏрђћрђ║
- **рђърђГрђ»рђюрђЙрђ▒рђгрђёрђ║рђЎрђЙрђ»рђћрђ▒рђЏрђг** - рђърђГрђ»рђюрђЙрђ▒рђгрђёрђ║рђЎрђЙрђ»рђћрђ▒рђЏрђгрђђрђГрђ» рђърђђрђ║рђърђгрђЁрђ▒рђЏрђћрђ║
- **рђЁрђћрђЁрђ║рђЁрђйрђЎрђ║рђИрђєрђ▒рђгрђёрђ║рђЏрђірђ║** - рђЁрђћрђЁрђ║рЂЈ рђЁрђйрђЎрђ║рђИрђєрђ▒рђгрђёрђ║рђЏрђірђ║рђђрђГрђ» рђЎрђ╝рђЙрђёрђирђ║рђљрђёрђ║рђЏрђћрђ║

### ­ЪћД рђЏрђЙрђёрђ║рђИрђюрђёрђ║рђИрђЎрђЙрђ»рђюрђ»рђЋрђ║рђєрђ▒рђгрђёрђ║рђЂрђ╗рђђрђ║рђЎрђ╗рђгрђИ (Cleanup Functions)

#### **рђАрђюрђГрђ»рђАрђюрђ╗рђ▒рђгрђђрђ║рђЏрђЙрђёрђ║рђИрђюрђёрђ║рђИрђЂрђ╝рђёрђ║рђИ (Automatic Cleanup)**
- **рђАрђЂрђ╗рђГрђћрђ║рђЄрђџрђгрђИ** - рЂЃ рђЏрђђрђ║рђљрђЁрђ║рђђрђ╝рђГрђЎрђ║ рђі рЂѓ рђћрђгрђЏрђ«рђљрђйрђёрђ║ рђАрђюрђГрђ»рђАрђюрђ╗рђ▒рђгрђђрђ║рђЏрђЙрђёрђ║рђИрђюрђёрђ║рђИрђЂрђ╝рђёрђ║рђИ
- **рђърђГрђЎрђ║рђИрђєрђірђ║рђИрђЎрђЙрђ»рђђрђгрђю** - рЂЃ рђЏрђђрђ║рђАрђљрђйрђёрђ║рђИ рђюрђ▒рђгрђирђЎрђ╗рђгрђИрђђрђГрђ» рђърђГрђЎрђ║рђИрђєрђірђ║рђИрђЂрђ╝рђёрђ║рђИ
- **рђАрђюрђГрђ»рђАрђюрђ╗рђ▒рђгрђђрђ║рђюрђ»рђЋрђ║рђєрђ▒рђгрђёрђ║рђЂрђ╝рђёрђ║рђИ** - рђюрђ░рђђрђ╝рђГрђ»рђИрђЁрђгрђИрђЎрђЙрђ»рђЎрђюрђГрђ»рђўрђ▓ рђАрђюрђГрђ»рђАрђюрђ╗рђ▒рђгрђђрђ║рђюрђ»рђЋрђ║рђєрђ▒рђгрђёрђ║рђЂрђ╝рђёрђ║рђИ

## рђАрђЎрђ╝рђћрђ║рђюрђ»рђЋрђ║рђєрђ▒рђгрђёрђ║рђЂрђ╗рђђрђ║рђЎрђ╗рђгрђИ (Quick Actions)

### ­ЪћД рђАрђЎрђ╝рђћрђ║рђюрђ»рђЋрђ║рђєрђ▒рђгрђёрђ║рђЂрђ╗рђђрђ║рђЎрђ╗рђгрђИрђђрђГрђ» рђАрђърђ»рђХрђИрђЋрђ╝рђ»рђЂрђ╝рђёрђ║рђИ (Using Quick Actions)

1. **рђЋрђёрђ║рђЎрђњрђђрђ║рђЏрђЙрђ║рђўрђ»рђљрђ║рђљрђйрђёрђ║ "рђАрђЎрђ╝рђћрђ║рђюрђ»рђЋрђ║рђєрђ▒рђгрђёрђ║рђЂрђ╗рђђрђ║рђЎрђ╗рђгрђИ" рђђрђЈрђ╣рђЇрђђрђГрђ» рђђрђ╝рђірђирђ║рђЏрђЙрђ»рђЋрђФ**
2. **рђюрђГрђ»рђАрђЋрђ║рђърђ▒рђг рђюрђ»рђЋрђ║рђєрђ▒рђгрђёрђ║рђЂрђ╗рђђрђ║рђђрђГрђ» рђЏрђйрђ▒рђИрђЂрђ╗рђџрђ║рђЋрђФ**
3. **рђюрђ»рђЋрђ║рђєрђ▒рђгрђёрђ║рђЂрђ╗рђђрђ║рђђрђГрђ» рђюрђ»рђЋрђ║рђєрђ▒рђгрђёрђ║рђЋрђФ**

## ­Ъџе рђАрђЏрђ▒рђИрђЋрђ▒рђФрђ║рђАрђЂрђ╝рђ▒рђАрђћрђ▒рђЎрђ╗рђгрђИ (Emergency Situations)

### Рџа№ИЈ рђАрђЏрђ▒рђИрђЋрђ▒рђФрђ║рђюрђ»рђЋрђ║рђєрђ▒рђгрђёрђ║рђЂрђ╗рђђрђ║рђЎрђ╗рђгрђИ (Emergency Actions)

#### 1. **рђЁрђћрђЁрђ║рђЋрђ╝рђ┐рђћрђгрђЎрђ╗рђгрђИ (System Issues)**
- **рђЁрђћрђЁрђ║рђђрђГрђ» рђЋрђ╝рђћрђ║рђюрђірђ║рђЁрђљрђёрђ║рђЂрђ╝рђёрђ║рђИ** - рђЁрђћрђЁрђ║рђЋрђ╝рђ┐рђћрђгрђЎрђ╗рђгрђИрђђрђГрђ» рђќрђ╝рђ▒рђЏрђЙрђёрђ║рђИрђЏрђћрђ║
- **рђњрђ▒рђљрђгрђўрђ▒рђирђЁрђ║рђђрђГрђ» рђЋрђ╝рђћрђ║рђюрђірђ║рђЁрђљрђёрђ║рђЂрђ╝рђёрђ║рђИ** - рђњрђ▒рђљрђгрђўрђ▒рђирђЁрђ║рђЋрђ╝рђ┐рђћрђгрђЎрђ╗рђгрђИрђђрђГрђ» рђќрђ╝рђ▒рђЏрђЙрђёрђ║рђИрђЏрђћрђ║
- **рђюрђ▒рђгрђирђЎрђ╗рђгрђИрђђрђГрђ» рђЏрђЙрђёрђ║рђИрђюрђёрђ║рђИрђЂрђ╝рђёрђ║рђИ** - рђЁрђћрђЁрђ║рђЁрђйрђЎрђ║рђИрђєрђ▒рђгрђёрђ║рђЏрђірђ║рђђрђГрђ» рђЎрђ╝рђЙрђёрђирђ║рђљрђёрђ║рђЏрђћрђ║

#### 2. **рђњрђђрђ║рђњрђюрђ▒рђгрђирђЋрђ╝рђ┐рђћрђгрђЎрђ╗рђгрђИ (Deadlock Issues)**
- **рђњрђђрђ║рђњрђюрђ▒рђгрђирђЎрђ╗рђгрђИрђђрђГрђ» рђЋрђ╝рђ▒рђюрђірђ║рђАрђ▒рђгрђёрђ║ рђюрђ»рђЋрђ║рђЂрђ╝рђёрђ║рђИ** - рђњрђђрђ║рђњрђюрђ▒рђгрђирђЎрђ╗рђгрђИрђђрђГрђ» рђЋрђ╝рђ▒рђюрђірђ║рђАрђ▒рђгрђёрђ║ рђюрђ»рђЋрђ║рђЂрђ╝рђёрђ║рђИ
- **рђњрђђрђ║рђњрђюрђ▒рђгрђирђЎрђ╗рђгрђИрђђрђГрђ» рђЁрђ▒рђгрђёрђирђ║рђђрђ╝рђірђирђ║рђЂрђ╝рђёрђ║рђИ** - рђњрђђрђ║рђњрђюрђ▒рђгрђирђЎрђ╗рђгрђИрђђрђГрђ» рђЁрђ▒рђгрђёрђирђ║рђђрђ╝рђірђирђ║рђЂрђ╝рђёрђ║рђИ

### ­ЪЊъ рђАрђЏрђ▒рђИрђЋрђ▒рђФрђ║рђєрђђрђ║рђърђйрђџрђ║рђЏрђ▒рђИ (Emergency Contacts)

#### **рђАрђђрђ║рђЎрђёрђ║рђАрђќрђйрђ▓рђи (Admin Team)**
- **рђАрђЊрђГрђђрђАрђђрђ║рђЎрђёрђ║** - рђЁрђћрђЁрђ║рЂЈ рђАрђЊрђГрђђрђАрђђрђ║рђЎрђёрђ║
- **рђћрђірђ║рђИрђЋрђірђгрђАрђђрђ║рђЎрђёрђ║** - рђћрђірђ║рђИрђЋрђірђгрђЋрђ╝рђ┐рђћрђгрђЎрђ╗рђгрђИрђАрђљрђйрђђрђ║
- **рђњрђ▒рђљрђгрђўрђ▒рђирђЁрђ║рђАрђђрђ║рђЎрђёрђ║** - рђњрђ▒рђљрђгрђўрђ▒рђирђЁрђ║рђЋрђ╝рђ┐рђћрђгрђЎрђ╗рђгрђИрђАрђљрђйрђђрђ║

## ­Ъћњ рђюрђ»рђХрђЂрђ╝рђ»рђХрђЏрђ▒рђИ (Security)

### ­ЪЏА№ИЈ рђюрђ»рђХрђЂрђ╝рђ»рђХрђЏрђ▒рђИрђюрђ»рђЋрђ║рђєрђ▒рђгрђёрђ║рђЂрђ╗рђђрђ║рђЎрђ╗рђгрђИ (Security Features)

#### 1. **рђАрђђрђ║рђЎрђёрђ║рђърђгрђђрђ╝рђірђирђ║рђЏрђЙрђ»рђћрђГрђ»рђёрђ║рђЂрђ╝рђёрђ║рђИ (Admin Only Access)**
- **рђЁрђћрђЁрђ║рђюрђ▒рђгрђирђЎрђ╗рђгрђИ** - рђАрђђрђ║рђЎрђёрђ║рђърђг рђђрђ╝рђірђирђ║рђЏрђЙрђ»рђћрђГрђ»рђёрђ║рђЂрђ╝рђёрђ║рђИ
- **рђЏрђЙрђёрђ║рђИрђюрђёрђ║рђИрђЎрђЙрђ»рђюрђ»рђЋрђ║рђєрђ▒рђгрђёрђ║рђЂрђ╗рђђрђ║рђЎрђ╗рђгрђИ** - рђАрђђрђ║рђЎрђёрђ║рђърђг рђюрђ»рђЋрђ║рђєрђ▒рђгрђёрђ║рђћрђГрђ»рђёрђ║рђЂрђ╝рђёрђ║рђИ
- **рђњрђђрђ║рђњрђюрђ▒рђгрђирђЎрђ╗рђгрђИ** - рђАрђђрђ║рђЎрђёрђ║рђърђг рђђрђ╝рђірђирђ║рђЏрђЙрђ»рђћрђГрђ»рђёрђ║рђЂрђ╝рђёрђ║рђИ

#### 2. **рђњрђ▒рђљрђгрђђрђгрђђрђйрђџрђ║рђЎрђЙрђ» (Data Protection)**
- **рђюрђ▒рђгрђирђЎрђ╗рђгрђИрђђрђГрђ» рђђрђгрђђрђйрђџрђ║рђЂрђ╝рђёрђ║рђИ** - рђюрђ▒рђгрђирђЎрђ╗рђгрђИрђђрђГрђ» рђђрђгрђђрђйрђџрђ║рђЂрђ╝рђёрђ║рђИ
- **рђЏрђЙрђёрђ║рђИрђюрђёрђ║рђИрђЎрђЙрђ»рђђрђГрђ» рђђрђгрђђрђйрђџрђ║рђЂрђ╝рђёрђ║рђИ** - рђЏрђЙрђёрђ║рђИрђюрђёрђ║рђИрђЎрђЙрђ»рђђрђГрђ» рђђрђгрђђрђйрђџрђ║рђЂрђ╝рђёрђ║рђИ
- **рђАрђЁрђ«рђЏрђёрђ║рђЂрђХрђЁрђгрђЎрђ╗рђгрђИрђђрђГрђ» рђђрђгрђђрђйрђџрђ║рђЂрђ╝рђёрђ║рђИ** - рђАрђЁрђ«рђЏрђёрђ║рђЂрђХрђЁрђгрђЎрђ╗рђгрђИрђђрђГрђ» рђђрђгрђђрђйрђџрђ║рђЂрђ╝рђёрђ║рђИ

## ­ЪЊъ рђђрђ░рђірђ«рђЏрђ▒рђИрђћрђЙрђёрђирђ║ рђЋрђХрђирђЋрђГрђ»рђИрђЎрђЙрђ» (Help and Support)

### ­Ъєў рђђрђ░рђірђ«рђЏрђ▒рђИрђћрђЙрђёрђирђ║ рђЋрђХрђирђЋрђГрђ»рђИрђЎрђЙрђ» (Help and Support)

#### **рђЋрђ╝рђ┐рђћрђгрђЎрђ╗рђгрђИрђђрђГрђ» рђќрђ╝рђ▒рђЏрђЙрђёрђ║рђИрђЂрђ╝рђёрђ║рђИ (Troubleshooting)**
- **рђюрђ▒рђгрђирђЎрђ╗рђгрђИрђђрђГрђ» рђђрђ╝рђірђирђ║рђЏрђЙрђ»рЂЇрђЎрђЏрђЂрђ╝рђёрђ║рђИ** - рђюрђ▒рђгрђирђЎрђ╗рђгрђИрђђрђГрђ» рђђрђ╝рђірђирђ║рђЏрђЙрђ»рЂЇрђЎрђЏрђЂрђ╝рђёрђ║рђИ
- **рђЏрђЙрђёрђ║рђИрђюрђёрђ║рђИрђЎрђЙрђ»рђюрђ»рђЋрђ║рђєрђ▒рђгрђёрђ║рЂЇрђЎрђЏрђЂрђ╝рђёрђ║рђИ** - рђЏрђЙрђёрђ║рђИрђюрђёрђ║рђИрђЎрђЙрђ»рђюрђ»рђЋрђ║рђєрђ▒рђгрђёрђ║рЂЇрђЎрђЏрђЂрђ╝рђёрђ║рђИ
- **рђњрђђрђ║рђњрђюрђ▒рђгрђирђЋрђ╝рђ┐рђћрђгрђЎрђ╗рђгрђИ** - рђњрђђрђ║рђњрђюрђ▒рђгрђирђЋрђ╝рђ┐рђћрђгрђЎрђ╗рђгрђИ

## ­ЪЊЮ рђћрђГрђѓрђ»рђХрђИ (Conclusion)

рђцрђЁрђћрђЁрђ║рђърђірђ║ рђѓрђГрђЎрђ║рђИрђЋрђюрђђрђ║рђќрђ▒рђгрђёрђ║рђИрЂЈ рђЁрђћрђЁрђ║рђюрђ▒рђгрђирђЎрђ╗рђгрђИрђћрђЙрђёрђирђ║ рђЏрђЙрђёрђ║рђИрђюрђёрђ║рђИрђЎрђЙрђ»рђђрђГрђ» рђЁрђ«рђЎрђХрђЂрђћрђирђ║рђЂрђйрђ▓рђЏрђћрђ║ рђЋрђ╝рђірђирђ║рђЁрђ»рђХрђърђ▒рђг рђЁрђћрђЁрђ║рђќрђ╝рђЁрђ║рђЋрђФрђърђірђ║рЂІ

- **рђюрђ▒рђгрђирђЎрђ╗рђгрђИрђђрђГрђ» рђЋрђ»рђХрђЎрђЙрђћрђ║рђЏрђЙрђёрђ║рђИрђюрђёрђ║рђИрђЋрђФ** - рђюрђ▒рђгрђирђЎрђ╗рђгрђИрђђрђГрђ» рђЋрђ»рђХрђЎрђЙрђћрђ║рђЏрђЙрђёрђ║рђИрђюрђёрђ║рђИрђЋрђФ
- **рђњрђђрђ║рђњрђюрђ▒рђгрђирђЎрђ╗рђгрђИрђђрђГрђ» рђЋрђ»рђХрђЎрђЙрђћрђ║рђЁрђ▒рђгрђёрђирђ║рђђрђ╝рђірђирђ║рђЋрђФ** - рђњрђђрђ║рђњрђюрђ▒рђгрђирђЎрђ╗рђгрђИрђђрђГрђ» рђЋрђ»рђХрђЎрђЙрђћрђ║рђЁрђ▒рђгрђёрђирђ║рђђрђ╝рђірђирђ║рђЋрђФ
